A simple light weight plugin to translate your WordPress website to some other language.

== Description ==

This is a light weight plugin which allows your customer to translate the website to their desire language. Basically, this plugin add a Google Transalate widget, which shows the language dropdown on the front end of website. Viewver can select any language from that drop down and all the content of your website will get translated to the selected language 

== Installation ==

Method 1:
<ol>
<li> Download the plugin wpgt-google-translate.zip </li>
<li> Unpack the wpgt-google-translate.zip file and extract the wpgt-google-translate folder </li>
<li> Upload the plugin folder to your /wp-content/plugins/ directory </li>
<li> Go to WordPress dashboard, click on Plugins from the menu </li>
<li> Locate the WordPress Google Translate and click on Activate link to activate the plugin. </li>
</ol>

Method 2:
<ol>
<li> Download the plugin wpgt-google-translate.zip </li>
<li> Go to WordPress Dashboard-Plugins-Add New </li>
<li> Click on Upload Plugin link from top </li>
<li> Upload the downloaded wpgt-google-translate.zip file and click on Install Now </li>
<li> After installation, click on Activate Plugin link to activate the plugin. </li>
</ol>
